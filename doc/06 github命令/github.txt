…或从命令行推送现有存储库
git remote add origin https://github.com/NewLifeReLoading/OKX-MT5-Arbitrage.git
git branch -M main
git push -u origin main

git add .  添加当前目录（包括子目录）的所有新文件、已修改文件、已删除文件（但不包括 .gitignore 中忽略的文件）。
git add -A 或 git add --all 效果与 git add . 类似，但会递归添加整个仓库的所有修改（即使当前不在仓库根目录）。
# 1. 修改文件后，查看工作区状态
git status  # 会显示“未暂存的修改”或“未跟踪的文件”

# 2. 将修改添加到暂存区
git add 被修改的文件

# 3. 再次查看状态，确认文件已进入暂存区
git status  # 会显示“ Changes to be committed ”

# 4. 提交暂存区内容到本地仓库
git commit -m "提交说明"

5 .强推到原远程，覆盖所有历史
git push -u origin main --force 强推到原远程，覆盖所有历史
